# ● 생성자

 -> 만들지 않으면 Compiler에서 자동으로 생성해준다.
 
Book.java
 
```java

public class Book {
	String title;
	String author;
	int price;
	
	public Book() {
		
	}
	public Book(String title) {
		this.title=title;
	}
	public Book(String title, String author) {
		this.author = author;
		this.title = title;
	}
	public Book(String title, String author, int price) {
		this.author = author;
		this.title = title;
		this.price = price;
	}
	
	public String getTitle() {
		return title;
	}
	public String getAuthor() {
		return title;
	}
	public int getPrice() {
		return price;
	}
}

```

main.java

```java
public class Main {
	public static void main(String[] args) {
			Book b1 = new Book("Book1","me",5000);
			
			Book b2 = new Book();
			
			System.out.println(b1.getPrice());
			
	}
	
}
```

### 생성자의 상속 super
super() : 상위 생성자 부르기
super.  : 상위 타입 부르기
Animal.java

```java
public class Animal {
	public Animal() {
		System.out.println("Making an Animal");
	}
}

```
Hippo.java

```java
public class Hippo extends Animal {
	public Hippo() {
		//super(); 이게 자동으로 들어가게됨 있어도 똑같고 없어도 똑같다.
		System.out.println("Making a Hippo");
	}
}

```
TestHippo.java

```java
public class TestHippo {
	public static void main(String[] args) {
		System.out.println("Starting.....");
		Hippo h = new Hippo();
	}
}
```

결과값은
```
animal이 먼저 출력되고
hippo가 출력된다.
```


### 생성자 this
this() : 현재 생성자 부르기
```java
public class Mini {
	String a;
	String b;
	String c;
	
	public Mini(String a) {
		this(a, "검정색","검정색"); //현재 클래스의 해당하는 인수의 클래스로
	}
	
	public Mini(String a, String b) {
		this(a, b,"검정색");
	}
	
	public Mini(String a, String b, String c) {
		this.a=a;
		this.b=b;
		this.c=c;
	}
		
}
```
Main.java
```java
public class Main {
	public static void main(String[] args) {
		Mini M = new Mini("빨간색");
		System.err.println(M.a + M.b );
	}
}
```
